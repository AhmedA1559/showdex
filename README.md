# `showdex`

[![Commitizen friendly](https://img.shields.io/badge/commitizen-friendly-brightgreen.svg)](http://commitizen.github.io/cz-cli/)

[Pokémon Showdown](https://play.pokemonshowdown.com) extension to help you drop them dracos.

> **NOTE**
>
> Currently only supports **Google Chrome**. Support for other major browsers is planned in the future.

*Showdex* integrates the [Pokémon Damage Calculator](https://calc.pokemonshowdown.com/) into the Pokémon Showdown client. The calculator's parameters automatically sync with your battle, so you can spend less time clicking and more time sweeping!

> More information coming soon!
